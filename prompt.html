For this problem, you will implement a function that swaps the values pointed to by two pointers. Because the function accepts void* pointers, you must use the provided type tags to determine the underlying data type before swapping values. See the behavior rules below for details. The function declaration should be in includes/solution.hpp, and the definition should be in src/solution.cc.

Define the following function:

bool SwapPointers(void* a, void* b, DataType typeA, DataType typeB);

The function should behave as follows:

  • If a == nullptr or b == nullptr, throw std::invalid_argument.
  • If typeA != typeB, return false and do not modify either value.
  • If typeA == typeB and the type is supported, swap the pointed-to values and return true.
  • If typeA == typeB and the type is unsupported, return false.

Supported types:

  • int
  • std::string

Important Constraints

  • Do not use templates.
  • Do not use dynamic_cast.
  • Do not dereference void* directly.
  • Use static_cast<T*> before dereferencing.
  • Swap the values *p1 and *p2, not the pointers a and b.
